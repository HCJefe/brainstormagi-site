# Brainstorm AGI — Corporate Website

> **[brainstormagi.com](https://brainstormagi.com)** — AI-powered automation for every industry.

The corporate website for Brainstorm AGI, an AI automation company based in Yuma, Arizona.

## Tech

Static site built with vanilla HTML, CSS, and JavaScript. Deployed via GitHub Pages.

## About Brainstorm AGI

We build multi-tenant AI agent platforms for industries still running on manual processes. Our current focus is home care operations — scheduling, caregiver management, compliance workflows, and intelligent communication agents (voice, SMS, email).

**Stack:** TypeScript / React / Express / PostgreSQL / AWS ECS Fargate / Terraform / Claude AI
